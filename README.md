# HigherMathematics

本仓库用于整理《大学数学 I》课程讲义，内容面向经济学、管理学、医学等非数学专业学生。讲义强调定义、定理、证明、例题与练习的完整衔接，同时加入数学文化阅读材料和机器学习中的导数应用示例。

## 仓库内容

- `大学数学I-完整讲义.tex`：合并后的完整 LaTeX 源码。
- `大学数学I-完整讲义.pdf`：当前已编译的完整讲义。
- `main.tex`：前半部分同学原始讲义源码。
- `大学数学I-导数与微分-中值定理讲义.tex`：导数、微分、中值定理及导数应用部分源码。
- `高数A习题课讲义.tex`：原讲义模板。
- `machine-learning-visualizations/`：讲义中机器学习相关例题的可交互可视化实现。

## 讲义结构

当前完整讲义包含：

1. 数学文化与数学史导引。
2. 数列、极限与无穷级数。
3. 函数连续性。
4. 导数与微分。
5. 中值定理及导数的应用。

其中数学史部分供学生课下阅读；正文部分尽量将关键定义、定理证明、例题、课后习题与参考答案组织在同一套格式中。

## 生成完整讲义

在仓库根目录运行：

```bash
xelatex -interaction=nonstopmode '大学数学I-完整讲义.tex'
xelatex -interaction=nonstopmode '大学数学I-完整讲义.tex'
```

第二次运行 `xelatex` 用于更新目录和交叉引用。

## 机器学习可视化示例

直接在浏览器中打开：

```text
machine-learning-visualizations/index.html
```

页面包含三个与讲义正文对应的课堂演示：

1. 讲义例 1：`L(w)=1/2(w-3)^2` 的一元梯度下降。
2. 讲义例 2：两个样本 `(1,2),(2,3)` 的线性回归损失和偏导数更新。
3. 讲义例 3：单样本平方损失 `L(w)=1/2(wx-y)^2` 的凸性、导数和二阶导数。
