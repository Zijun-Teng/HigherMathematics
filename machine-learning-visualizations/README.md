# 讲义机器学习例题可视化

这是配合《大学数学I》讲义中机器学习相关例题的课堂演示。示例使用纯 HTML/CSS/JavaScript 编写，不需要安装依赖。

## 使用方法

直接在浏览器中打开 `index.html` 即可。若要放到 GitHub Pages，可以把本文件夹提交到仓库，并将 Pages 的入口指向该目录或将 `index.html` 移到仓库根目录。

## 示例内容

GitHub 仓库首页会直接显示 `preview.svg` 静态预览图；交互版在 `index.html` 中。

1. 讲义例 1：`L(w)=1/2(w-3)^2` 的一元梯度下降。
2. 讲义例 2：两个样本 `(1,2),(2,3)` 的线性回归损失和偏导数更新。
3. 讲义例 3：单样本平方损失 `L(w)=1/2(wx-y)^2` 的凸性、导数和二阶导数。

## 课堂提示

梯度下降的核心公式为

```text
theta_{k+1} = theta_k - eta * grad L(theta_k)
```

其中 `eta` 是学习率。学习率过大时，迭代可能震荡甚至发散；学习率过小时，下降很稳定但速度较慢。
