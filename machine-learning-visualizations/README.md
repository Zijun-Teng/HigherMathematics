# 机器学习中的导数与梯度下降可视化

这是配合《大学数学I》导数、偏导数与凸性内容的课堂演示示例。示例使用纯 HTML/CSS/JavaScript 编写，不需要安装依赖。

## 使用方法

直接在浏览器中打开 `index.html` 即可。若要放到 GitHub Pages，可以把本文件夹提交到仓库，并将 Pages 的入口指向该目录或将 `index.html` 移到仓库根目录。

## 示例内容

1. 一元梯度下降：展示损失函数曲线、当前位置、切线方向和迭代轨迹。
2. 线性回归拟合：展示参数 `w,b` 通过偏导数更新后，拟合直线逐步接近数据点。
3. 凸性与非凸性：比较凸函数与非凸函数上梯度下降的不同表现，帮助理解“局部极小”和“全局极小”的区别。

## 课堂提示

梯度下降的核心公式为

```text
theta_{k+1} = theta_k - eta * grad L(theta_k)
```

其中 `eta` 是学习率。学习率过大时，迭代可能震荡甚至发散；学习率过小时，下降很稳定但速度较慢。
