导数、偏导数与梯度下降

这页演示把讲义中的导数概念放进机器学习语境：导数给出一元函数的下降方向，偏导数给出多参数模型的更新方向，凸性帮助判断优化问题的整体结构。

一元梯度下降

损失函数取 \(L(w)=\frac12(w-3)^2+0.4\)。红点表示当前参数，蓝线表示当前位置的切线。

线性回归中的偏导数

模型为 \(\hat y=wx+b\)，损失函数为所有样本平方误差的平均值。

  • w
  • b
  • 损失
  • 迭代次数

dL/dw = (2/m) Σ(wx_i+b-y_i)x_i
dL/db = (2/m) Σ(wx_i+b-y_i)

偏导数分别衡量损失对 w 和 b 的敏感程度，两个参数同步沿负梯度方向更新。

凸函数与非凸函数

同一个算法在不同形状的损失函数上表现不同。凸函数更容易保证全局最优，非凸函数可能受初始点影响。
