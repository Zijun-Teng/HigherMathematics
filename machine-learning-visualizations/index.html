讲义机器学习例题可视化

本页只实现讲义中新增的机器学习相关例题：一元梯度下降、两样本线性回归的偏导数更新、单样本平方损失的凸性。每个模块都对应讲义中的公式，方便课堂演示或学生课下自查。

讲义例 1：一元梯度下降

讲义例题：设损失函数 \(L(w)=\frac12(w-3)^2\)，用 \(w_{k+1}=w_k-\eta L'(w_k)\) 讨论 \(0<\eta<2\) 时的收敛性。

  • w_k
  • L(w_k)
  • L'(w_k)
  • 迭代次数

L'(w)=w-3
w_{k+1}-3=(1-eta)(w_k-3)

讲义例 2：两个样本的线性回归

讲义例题：对两个样本 \((1,2),(2,3)\)，线性模型 \(\hat y=wx+b\) 的损失函数为 \(J(w,b)=\frac14[(w+b-2)^2+(2w+b-3)^2]\)。

  • w
  • b
  • J(w,b)
  • 梯度
  • 迭代次数

dJ/dw=1/2[(w+b-2)+2(2w+b-3)]
dJ/db=1/2[(w+b-2)+(2w+b-3)]

初始点为 \((0,0)\) 时，讲义计算得到 \(\nabla J(0,0)=(-4,-5/2)\)，第一次更新为 \((4\eta,5\eta/2)\)。

讲义例 3：单样本平方损失的凸性

讲义例题：设 \(L(w)=\frac12(wx-y)^2\)，其中 \(x,y\) 为常数，讨论 \(L\) 关于 \(w\) 的凸性。

  • L(w)
  • L'(w)
  • L''(w)
  • 最小点

L'(w)=x(wx-y)
L''(w)=x^2 >= 0
